# Каскада Маркет

**Каскада Маркет** — маркетплейс локальных брендов, малых производств и ремесленников с реферальной моделью товарооборота.

## Базовая экономика

- Комиссия платформы с успешной продажи: **20%**
- Продавцу начисляется: **80%**
- Реферальная сеть: **10%**
- Фонд платформы: **10%**
- Подписка продавца: **1490 ₽/мес с 3-го месяца**

## Реферальные выплаты

| Линия | Процент |
|---|---:|
| 1 линия | 5% |
| 2 линия | 2.5% |
| 3 линия | 1.5% |
| 4 линия | 0.5% |
| 5 линия | 0.3% |
| 6 линия | 0.2% |

## MVP-модули

- Витрина и каталог товаров
- Карточка бренда и товара
- Кабинет партнёра
- Кабинет продавца
- Финансовый расчёт комиссии
- Демонстрационная админ-панель

## Локальный запуск

```bash
npm install
npm run dev
```

Открыть: http://localhost:3000
